## üõ†Ô∏è Getting Started

### Prerequisites

To run the programs in this repository, ensure you have the following installed:

- [Python 3.x](https://www.python.org/)
- Necessary AI/ML libraries (Pandas, NumPy, Scikit-learn, etc.)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/toxicbishop/Artificial-Intelligence-For-Business-AIB.git
   ```

2. Navigate to the desired program folder:

   ```bash
   cd AI-For-B/01
   ```

3. Run the script:

   ```bash
   python 01.py
   ```
